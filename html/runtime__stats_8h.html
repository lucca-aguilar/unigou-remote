Detailed Description

This module provides the interface to configure and read high-resolution timer metrics required by FreeRTOS to calculate CPU usage per task.

Note
This file utilizes extern "C" linkage to ensure compatibility between FreeRTOS kernel hooks and the Arduino framework.

void runtime_stats_init ( )

Initializes the runtime statistics subsystem and its reporting mechanism.

Warning
This function should only be called once, during system initialization and after the logging subsystem has been initialized. If there is a failure in creating the task for processing runtime statistics, the system will display an error message and enter an infinite loop.
