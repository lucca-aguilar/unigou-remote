◆ log_init()

void log_init ( )

Initializes the log queue and the log processing task.

Warning
This function should only be called once, during system initialization and before any call to log_write(). If there is a failure in allocating memory for the log queue or in creating the task for processing the logs, the system will display an error message and enter an infinite loop.
