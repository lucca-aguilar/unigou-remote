Initializes the memory monitor module and spawns its reporting task. 

\begin{DoxyWarning}{Warning}
This function should only be called once, during system initialization and after the logging subsystem has been initialized. If there is a failure in creating the task for processing memory data, the system will display an error message and enter an infinite loop. 
\end{DoxyWarning}
